HOW TO GET 60fps battles 
For emulator users:

1. ULJS00293.ini goes in the memstick/cheats folder or, if on Android, the psp/cheats folder.

2. To enable cheats, bring up the PPSSPP menu.

3. Click Settings, then the System tab, scroll down until you get to the Cheats section (the second one from the bottom). Click "Enable Cheats".

4. Now go back to the PPSSPP menu and you will now have an option called "Cheats". Click it, and enable the 60fps cheat.

For PSP/PSVita users:

1. You need the TempAR plugin: https://github.com/raing3/tempar/releases/latest To be able to see the full cheat list you need the latest version of TempAR (>=1.6.4).

2. Put ULJS-00293.db inside /SEPLUGINS/TempAR/cheats

3. When inside the game you'll need to press R+HOME to open TempAR, the tab should be [Cheater].
   Select the cheats you want to turn on with X.
   If you want a cheat to always be ON, modify its name in ULJS-00293.db and change "_C0" to "_C2".

4. Press NOTE (♪) to turn the Cheat Engine on.
   Navigate to the [PRX] tab by pressing R and set "Max Cheats?" to 2000 and "Max Blocks?" to 2496
   then select "Save Settings". 

5. Press O to exit the TempAR menu and go back to the game.

How to enable local multiplayer 
The cheat code needed to run local coop is included with the cheat ini file.
You'll need a specialized version of ppsspp to actually make use of it, though.
You can find instructions on how to set it up along with a link to the special ppsspp build here:
https://github.com/ucuckic/tales_patches/tree/main/ppsspp

Again, the cheat code needed to run this is in the cheat file included with the patch.
You don't need to mess with creating a new ini file or pasting code or anything like that.
If you want to use the other cheat codes on the site, you can paste them into the ini, however.
